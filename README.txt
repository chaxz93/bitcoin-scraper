The QR Scanner requires https://www.apachefriends.org/index.html and https://github.com/schmich/instascan
Your <sample_filename>.html code and instascan.min.js file (from the 'minified' version of Instascan) needs to be in the \XAMPP\htdocs directory of your XAMPP installation. From XAMPP's control panel start the 'Apache' server and browse to localhost:80/<sample_filename>.html to run your code.
